# Project Name

This repository contains a prototype of a Unity-based editor for semi-automatic level and world generation for various game types, particularly top-down roguelikes. The project focuses on procedural content, gameplay systems, and scalability.
The project focuses on procedural content, gameplay systems, and scalability.

---

## About

This project is still **in development** and will be gradually expanded for future projects/games.  
Its main goals are:

- experimenting with procedural level generation
- designing scalable enemy and difficulty systems
- building a solid technical foundation for a roguelike-style game

The project is intended for educational and experimental purposes, but it can
be freely reused and modified according to the license.

---

## Features

- Semi-automatic level/room generation
- Manually modify generated levels
- Scalable enemy parameters (damage, health, speed, armor, etc.)
- Difficulty calculation based on player power
- Data configuration based on ScriptableObject and external text files 

---

## Requirements

- **Unity version:** 2019.4.14f1  
- **Target platform:** PC (Windows / macOS / Linux)

> The project may not work correctly on older Unity versions.

---

## How to Run

1. Clone or download this repository
2. In the Assets folder, unzip the Textures.zip, Sprites.zip and Prefabs.zip files
3. Open **Unity Hub**
4. Add the project using the repository folder
5. Configuration files contained in the LevelEditor2D folder should be located in the path C:\Users\USER\AppData\LocalLow\CompanyName\ProductName\ 
where CompanyName and ProductName are set in the project settings (Player)
